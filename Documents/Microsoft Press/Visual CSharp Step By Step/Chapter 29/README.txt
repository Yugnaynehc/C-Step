To use the completed ProductDetailsService and ProductInformationService services:

1. Rename the ProductDetailsService - Complete folder to ProductDetailsService
2. Rename the ProductInformationService - Complete folder to ProductInformationService
3. Rebuild the ProductDetailsContracts - Complete project
4. Open the ProductDetailsService solution and add a reference to the ProductDetailsContracts assembly in the ProductDetailsContracts - Complete\ProductDetailsContracts\bin\Debug folder

To use the completed ProductClient application

1. Rebuild the ProductDetailsContracts - Complete project
2. OPen the ProductClient - Complete project and add a reference to the ProductDetailsContracts assembly in the ProductDetailsContracts - Complete\ProductDetailsContracts\bin\Debug folder
